# Project HA Docs

Project HA is organized into four root folders only:

- `backend`: Spring Boot REST API, PostgreSQL migrations, SMTP and R2 integration.
- `mobile-flutter`: Flutter app using REST APIs and secure token storage.
- `admin-web`: React/Vite admin panel using backend APIs only.
- `docs`: architecture, API, database, migration and local development notes.

Start with `docs/local-development.md`.
